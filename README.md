# 🚀 ROWAIM X - Ultimate Video & Audio Downloader

**ROWAIM X** ek powerful aur fast Command Line Interface (CLI) tool hai jo YouTube (Shorts & Long), Instagram Reels, Facebook, aur 1000+ websites se videos aur audio download karne ke liye banayi gayi hai.

---

## ✨ Features
- 📥 **Video Download:** Best quality MP4 videos download karein.
- 🎵 **Audio Extraction:** Kisi bhi video ko direct MP3 mein convert karein.
- ⚡ **Live Progress:** Download speed, percentage, aur ETA (time) live dekhein.
- 📦 **Bulk Downloading:** Ek saath 10-20 links paste karein aur tension free ho jayein.
- 📂 **Auto-Directory:** Files khud-ba-khud `/sdcard/Download/ROWAIM_X_Downloads` mein save hoti hain.
- 🎨 **Stylish UI:** Colorful terminal interface aur bara "ROWAIM X" banner.

---
